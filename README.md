# SafeQueue - Industrial Content Moderation Infrastructure

SafeQueue is a premium, high-precision moderation system design portfolio. It is built for Trust & Safety professionals, platform architects, and AI/human hybrid workflow designers.

## Technical Specifications

- **Aesthetic**: Industrial-Utilitarian (Monochromatic, Data-Dense, Sharp Geometry)
- **Theme**: Pure Dark (#000000) and Pure Light (#FFFFFF)
- **Typography**: IBM Plex Mono (Technical/Data) & Epilogue (Body)
- **Frameworks**: None (Vanilla HTML5, CSS3, ES6+ JS)
- **Icons**: Phosphor Icons (CDN)
- **RTL Support**: Full logical properties and `rtl.css` overrides
- **Responsive**: 360px to 1440px+

## Features

1. **Mission-Critical Design**: Every element communicates reliability and scale.
2. **Hybrid Dashboard**: A unified view of KPIs, Queue Management, Rule Engines, and Analytics.
3. **Advanced Animations**: CSS-only queue pipeline and JS typewriter effects.
4. **Perfect Dark/Light Mode**: Strictly `#000` and `#FFF` with system preference detection and persistence.
5. **RTL Logic**: Flipped layouts with logical property support.
